MATLAB Installation

Date Sat 02 September 2017

You can acquire a MATLAB license and instructions for installing the software by sending an email to askit@ldeo.columbia.edu. MATLAB uses a license server on campus and thus may only run when your computer is on the university network, or if you remotely connect using the LDEO VPN server.
