Date Thu 28 September 2017 Tags fortran

Basic structure

Programs

Make a file named HelloWorld.f90 that has the following content:

program HelloWorld
  write(*,*) 'Hello World'
end program HelloWorld

You can compile the code by

$ gfortran HelloWorld.f90 -o HelloWorld

Run the code in the terminal

$ ./HelloWorld
Hello World
